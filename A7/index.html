Attributes vs Properties

Understanding the difference between HTML attributes and DOM properties.

HTML Attribute

Set in HTML markup. Stays fixed — doesn't update when user types.

input.getAttribute("value")
VS

DOM Property

Live JavaScript value. Updates in real-time as user types.

input.value

Data Attributes Demo

Every task card uses data-* attributes for ID, status and category.

card.setAttribute("data-id", task.id);
card.setAttribute("data-status", "pending");
card.setAttribute("data-category", task.category);

// Read back
card.getAttribute("data-id");   // string
card.dataset.status;            // shortcut via dataset

Browser Rendering Pipeline

How the browser turns HTML + CSS into pixels on your screen.

HTML
Parsing

Browser reads HTML byte by byte, character by character.

Tokenization

HTML is split into tokens: tags, text, attributes.

DOM Tree

Tokens become nodes linked in a tree structure.

+
CSS
CSS Parsing

Browser reads and parses all stylesheet rules.

CSSOM Tree

CSS rules become a tree with computed styles for each node.

=
OUTPUT
Render Tree

DOM + CSSOM combined. Only visible elements included.

Layout

Browser calculates exact position and size of each element.

Paint

Fills in pixels: colors, borders, text, shadows.

Composite

All layers merged → final image shown on screen.
